8 Image Freespin Spinner

Functionality overview

The 6 image snapspin module was custom designed and built for the RAV4 2015 brochure. The module consists of a circular image rotating as the user swipes on the screen. When the user swipes the wheel spins freely slowing down realistically to a stop at one of the segments. This module features 8 content areas allowing for different content in each segment. The 8 spin features only one layer of fixed content.

The 8 image snapspin module has been used in the RAV4 brochure to display its ability to transition from different environments. The module itself has 8 segments on the module to utilise although the RAV4 model only uses 4 and repeats the others. With a small piece of simple text content the module can be used for various different scenarios. Although the module itself doesn't take very long to populate into the app itself the production time to produce the imagery required can take longer varying depending on complexity and effects.
