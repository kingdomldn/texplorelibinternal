Zoom Hotspots Configurator

Functionality overview

The Zoom configurator adds a dynamic effect to the standard Hotspot based configurator module. When the user taps on a hotspot the hotspot fades away as the user is zoomed into the relevant part of the image. This zoom effect transitions into the related module that has been associated to the hotspot selected.

The configuration variations on this module are only limited by what you can filter in a one way direction. Since this unit allows for hotspots the functionality is much larger than a config variant with no hotspots. These hotspots can link to any other existing module. Currently it is mostly used to load an image over the top of the unit although this could be swapped for a video, a text and image 360. The options are only limited by the content production.
