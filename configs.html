Configurator Modules

Configurator modules add a level of complexity to the application. These experiences offer the user the choice of the type of content they view. Allowing the user to choose from a varying degree of stacked options. An example might be the colour and grade of the car. These modules have a great flexibility as they allow us to define set content groups that you may wish the user to see, or allow the user to compare content in a digestible app experience instead of on the website.

The app currently has the functionality for 3 variants of Configurator modules, No hotspots, Zoom Hotspots &Standard Hotspots. Each of these variants allows for varying complexities of configuration depending on the content available to be generated for the vehicle. Examples used in the app currently are the configuration of a car model through both its interior and exterior images.
