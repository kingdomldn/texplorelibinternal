360° 3D Modules

360° 3D modules allow for a rich user driven experience for a car. The module features a realistic 3D render of the object your are trying to display. The user is then able to spin this object round in 3D across 36 frames of imagery. This module works well with a full 3D render of the car itself as it allows the user to effectively walk around the car.

Currently the 360° 3D module only exists within one variant which allows for a full screen experience. This allows for a wide variety of content. Any scene or object that can be rendered in 3D is valid for the app. This has scene versions of the module being used for studio shots of the car all the way to street scenes rendered around the car. With the addition of Hotspots on the 360° 3D module a further layer of information can be added to an experience.
