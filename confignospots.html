No Hotspots Configurator

Functionality overview

The standard configurator module is modifiable to not include any hotspots. This includes all the same customisable options that give the module a layer of complexity but without having to add any extra levels of information as the hotspots add. This can both save time on content creation and population. Allowing the user to see various types of visual information without having to leave the module.

This module types will work with any content that requires either customising or has multiple levels. Currently the module has been used to display different grades and colours of an exterior. This module does allow for a large variation of content, for example one tab of options could change the exterior colour whilst another allows the user to take a peek inside the car. A further option may be to extend the module to display customisation of other parts of the car such as the interior display units or fabric interiors.
