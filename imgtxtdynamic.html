Dynamic Content Text & Image

Functionality overview

The Dynamic text and image module gives the end user the opportunity to digest a larger piece of content in comparison to a full screen image module. With options akin to the configurator modules this gives the end user options to switch the content displayed in the module. Using a familiar content device of the base button bar the user is able to pick from one set of options or multiple. The module itself consists of two content areas. A large image area on the left hand side and a text container on the right hand side. The text area has options for a title, a subtitle & a body text area.

Options when using this module

Because of the base bar the end user has options to switch associated content using various factors. The module can be used to display just one piece of switchable content for example one set of engines, or the module can be adapted to support configuration options such as two different types of engines (hybrid & standard). This creates a module that is not restricted and allows for deep levels of information.
