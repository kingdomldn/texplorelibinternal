Toyota Explore Module Library

Full Screen 360° 3D

Functionality overview

The full screen 360° 3D module allows for the user to swipe left or right and rotate a 3D object in 360°. Over a 36 frame sequence we can place hotspots anywhere in the scene that link to any of the existing module types. This module is based on 3D rendered imagery the output displayed in the app is anything that can be created by the 3D team at Toyota.

Options when using this module

The 360° modules should be used as a base module that is launched from a page similar to that of a configurator. The module has the option of displaying hotspots or just featuring a 3D asset. When applying hotspots to the module any of the existing modules can be attached. In the videos above the image & text module has been used although this can be changed, some effective alternatives may be a video player or image(s) modules. Please note that the number of hotspots in the module will affect the amount of time taken to populate the content into the application.
