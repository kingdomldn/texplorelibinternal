Toyota Explore Module Library

Video Player

Functionality overview

The video player in Toyota explore launches straight into the video when a page is detected. Playing the video full screen with audio (if the device isn't muted). When the video has ended the user is given the option to replay the video or return to their previous location in the app.

Options when using this module

Since the video player module is a self contained experience it can be used anywhere in the app that it can be launched from. The most basic use is by scanning a page to launch the video. Alternative options are whenever a standard hotspot is used in an experience. An example of this could be the user tapping on a hotspot in a config module to load a video then return to the config unit.
