Text & Image Modules

Text & image modules are great for displaying both rich media and information at the same time. This can take the form of static content, where the frame is fixed with one piece of content. Or via a mixture of dynamic menu based content to change the content being displayed.

When creating text and image content it is best to consider the likely function of each module type. The static content option is best used when loading off of a hotspot. This allows the user to digest a small piece of text quickly before returning back to the main module. When using the dynamic option this is best used as a full module as the user will potentially have various levels of content to digest.
