Toyota Explore Module Library

Module Library Overview

Experience modules for Toyota Explore are designed to be reusable and adaptable to many situations car teams may find themselves requiring. On this page you will find a full list of all the modules currently available in the app, Categorised by type and variants. When selecting modules for use within a brochure package be aware that combinations of modules are possible. These combinations will be discussed on individual module profiles.

An example of this combination is that a Configuration module allows for each hotspot to go to a different piece of content. One hotspot may load a video whilst another may load an image, Stacking up of module types will create a differing experience between cars and give the user varying forms of exciting content to consume.

Configurators:

Configurator modules allow the user to change the content displayed in the device based on varying levels of selections. Config experiences have the option to include hotspots overlaid onto the imagery.
