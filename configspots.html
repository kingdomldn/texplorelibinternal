Standard Hotspots Configurator

Functionality overview

Configurator units are one of the more complex modules included within the Toyota Explore app. The module allows the user to filter through a list of options, each selection in these options changes the output displayed on the screen. These operate in a similar manner to the online configurators. Allowing a one way direction of filter traffic, For example if the unit was displaying exterior car options the user would start by selecting a car type (standard or hybrid) then a car grade option and finally a colour. The standard configuration module also allows for the addition of hotspots on top of the image. When selected these load another module over the top.

The configuration variations on this module are only limited by what you can filter in a one way direction. Since this unit allows for hotspots the functionality is much larger than a config variant with no hotspots. These hotspots can link to any other existing module. Currently it is mostly used to load an image over the top of the unit although this could be swapped for a video, a text and image 360. The options are only limited by the content production.
