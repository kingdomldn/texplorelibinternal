Toyota Explore Module Library

Full Screen images

Functionality overview

The full screen image module simply loads an image defined in the app filling the devices display. The module has basic attributes to define titles and has potential to display animated images by using an animated gif file.

Options when using this module

When using the image module it is worth bearing in mind that the module is self contained and has no options to change content. As such it is best to use this module as a piece of hotspot content. When populating content it is possible to add a title to each image in order to give some definition and further structure. Since the background of this module is just a flat image it is possible to add further content so long as it is baked into the image itself, this creates further production times but does allow for lots of content variation.
